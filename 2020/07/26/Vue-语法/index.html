Vue 使用HTML的基本语法。官网一句话

在底层的实现上，Vue 将模板编译成虚拟 DOM 渲染函数。结合响应系统，Vue 能够智能地计算出最少需要重新渲染多少组件，并把 DOM 操作次数减到最少。

插值

文本插值

<span v-once>Message: {{ msg }}</span>

将 msg 绑定到数据对象中，使用v-once 表示该节点上的数据绑定只渲染一次。如果数据对象中的值发生改变，将不会更新。

解析html

<span v-html="msg"></span>

data () {
  msg： "This is a Msg:<h1>HELLO!</h1>"
}

数据中的msg将会以html的方式插入到节点中，h1标签将会被浏览器解析

属性

使用 v-bind 将可以向节点传值，绑定数据

<div :id="item.id"></div>

HTML中使用js

<span>{{ dialogStatus == 'Create' ? 'Creating' : 'Updating' }}</span>

指令

参数

在 Vue-起步 中提到过使用v-开头定义的特殊属性

使用v-bind绑定数据

使用v-model 双向绑定数据

使用v-on 监听事件

动态参数

今天回溯了一遍Vue的官方教程文档，发现了 2.6.0 以上的版本新增的功能，可以使用动态的参数名

<a v-bind:[attributeName]="msg"> ... </a>
<a v-bind:[attributeName2]="method2"> ... </a>


data () {
  attributeName: 'value',
  attributeName2: 'click',
  msg： "This is a Msg."
}

method: {
  method2 () {
    console.log('hello~')
  }
}

修饰符

以.作为标识特殊后缀。

<button @click.stop="saySomthing(msg)">hello</button>

例如上述的修饰符，阻止点击事件冒泡
